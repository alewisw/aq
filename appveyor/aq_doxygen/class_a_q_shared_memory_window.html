Detailed Description

Provides a limited window into a shared memory region.

This is effectivy a subregion of an existing shared memory region.

Constructs a new AQSharedMemoryWindow object that provides a window into another shared memory object.

Parameters
smThe other shared memory object. The sm object must remain valid and in scope so long as this object is valid and in scope.
offThe offset into the other shared memory object.
sizeThe size of the region in the other shared memory objet to provide a window into.
Exceptions
std::out_of_rangeIf off is greater than or equal to the size of the passed shared memory region.
std::length_errorIf off plus size is greater than the size of the passed shared memory region.
